snapcraft
user manual — keyboard shortcuts & tools
screenshot studio
v1.0 · 2026
01 — capture
⌘ + ⌥ + 4
captured area
drag to select any region of the screen. releases the capture editor.
02 — annotation tools
T
text tool
add typed labels & callouts
A
arrow tool
draw directional arrows
P
pen tool
freehand drawing & sketching
H
highlight tool
semitransparent color wash
N
step pin tool
numbered markers for walkthroughs
B
blur tool
redact / censor sensitive areas
O
ocr tool
extract text from any screen region
R
rectangle tool
draw outline or filled boxes
L
line tool
straight lines & dividers
C
crop tool
trim the captured region
V
select tool
move & resize existing annotations
esc
quit current tool
exits active tool → falls back to select
03 — output
⌘ + C
copy to clipboard
copies the annotated screenshot as a PNG. paste anywhere.
